# 🍽️ DineSync

DineSync is a modern restaurant management and food ordering platform designed to simplify the dining experience for both customers and restaurant administrators. It provides an intuitive interface for browsing menus, placing orders, and managing restaurant operations efficiently.

---

## ✨ Features

### 👤 Customer
- User Registration & Login
- Browse Food Menu
- Search & Filter Food Items
- Add Items to Cart
- Place Orders
- View Order History
- Responsive User Interface

### 👨‍💼 Admin
- Secure Admin Authentication
- Add, Update, and Delete Menu Items
- Manage Customer Orders
- Update Order Status
- Dashboard for Restaurant Management

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/safeenaks/Dinesync.git
cd Dinesync
```

---

### Backend Setup

```bash
cd server
npm install
```

Create a `.env` file inside the `server` directory.

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET_KEY
```

Start the backend server:

```bash
npm run dev
```

---

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
